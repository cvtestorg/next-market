# 用户角色与权限

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 终端用户

### 1.1 插件使用权限申请

| 场景 | 说明 |
|------|------|
| 申请理由 | 用户填写申请使用该插件的理由 |
| 申请时间 | 记录申请提交的时间 |
| 状态跟踪 | 用户可查看申请的审核状态 |
| 审核结果 | 通过/拒绝，拒绝需提供原因 |
| 重新申请 | 被拒绝后可修改理由重新申请 |

### 1.2 插件管理操作

| 操作 | 前置条件 | 说明 |
|------|----------|------|
| 安装插件 | 有使用权限 | 将插件安装到个人环境 |
| 卸载插件 | 已安装 | 从个人环境移除插件 |
| 购买插件 | 收费插件 | 支付后获得使用权限 |
| 设置个人配置 | 插件有个人配置项 | 配置插件的个人参数 |
| 启动/停止插件 | 已安装 | 控制插件的运行状态 |
| 更新插件 | 有新版本 | 升级到最新版本 |

### 1.3 插件上传（开发者角色）

| 功能 | 说明 |
|------|------|
| 上传插件 | 提交插件包等待审核 |
| 管理版本 | 发布新版本、查看历史版本 |
| 管理权限 | 设置插件的可见范围和使用权限 |
| 设置价格 | 收费插件设置售价 |
| 查看统计 | 查看安装量、使用情况等 |

## 2. 企业管理员

| 功能 | 说明 |
|------|------|
| 审核权限申请 | 审批终端用户的插件使用权限申请 |
| 上架局域插件 | 将企业内部插件上架到企业市场 |
| 分配权限 | 批量分配插件使用权限给用户/部门 |
| 设置企业配置 | 配置插件的企业级参数 |
| 管理局域插件 | 管理企业内部插件的版本和状态 |

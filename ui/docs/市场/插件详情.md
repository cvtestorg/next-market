# 插件详情

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 插件图标 | 插件的 Logo 图标 | 是 |
| 插件名称 | 插件的显示名称 | 是 |
| 插件描述 | 富文本格式，支持图文混排 | 是 |
| 插件分类 | 插件所属的分类标签 | 是 |
| 插件版本 | 当前最新版本号 | 是 |
| 更新日志 | 各版本的更新说明 | 是 |

## 2. 作者信息

| 字段 | 说明 |
|------|------|
| 作者头像 | 作者的头像图片 |
| 作者名称 | 作者的显示名称 |
| 认证标识 | 官方认证/企业认证/个人开发者 |
| 其他作品 | 跳转到该作者的其他插件列表 |

## 3. 插件操作

| 操作 | 说明 | 权限要求 |
|------|------|----------|
| 安装 | 将插件安装到当前环境 | 拥有使用权限 |
| 卸载 | 从当前环境移除插件 | 已安装该插件 |
| 启动 | 启用已安装的插件 | 已安装该插件 |
| 停止 | 禁用已安装的插件 | 已安装该插件 |
| 删除 | 从市场中删除插件（仅作者） | 插件作者 |
| 更新 | 升级到最新版本 | 已安装且有新版本 |
| 购买 | 购买收费插件的使用权限 | 收费插件未购买 |

## 4. 插件配置

### 4.1 配置类型

| 配置类型 | 说明 | 配置者 |
|----------|------|--------|
| 企业配置 | 企业级别的统一配置 | 企业管理员 |
| 个人配置 | 用户个人的私有配置 | 终端用户 |

### 4.2 配置项属性

| 属性 | 说明 | 必填 |
|------|------|------|
| 配置项名称 | 配置项的显示名称 | 是 |
| 配置项类型 | 文本/单选/多选/日期/时间/数字/布尔/密码 | 是 |
| 配置项Key | 配置项的唯一标识符 | 是 |
| 配置项选项 | 选择类型的可选值列表 | 否（选择类型必填） |
| 配置项描述 | 配置项的说明文字 | 否 |
| 配置项默认值 | 配置项的默认值 | 否 |
| 配置项验证规则 | 输入值的验证规则 | 否 |
| 配置项示例 | 配置项的示例值 | 否 |

### 4.3 配置项类型说明

| 类型 | 说明 | 输入组件 |
|------|------|----------|
| 文本 | 普通文本输入 | Input |
| 单选 | 单选项 | Radio / Select |
| 多选 | 多选项 | Checkbox / MultiSelect |
| 日期 | 日期选择 | DatePicker |
| 时间 | 时间选择 | TimePicker |
| 数字 | 数字输入 | NumberInput |
| 布尔 | 开关选择 | Switch |
| 密码 | 密码输入（加密存储） | Password |

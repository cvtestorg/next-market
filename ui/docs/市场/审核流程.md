# 审核流程

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 审核状态

| 状态 | 说明 | 后续操作 |
|------|------|----------|
| 审核中 | 等待审核方处理 | 等待 |
| 已认证 | 审核通过，已上架 | 可正常使用 |
| 被拒绝 | 审核未通过 | 查看原因，可重新提交 |
| 已下架 | 主动或被动下架 | 重新上架或删除 |

## 2. 插件上架审核流程

### 2.1 个人插件上架流程

```
终端用户提交插件 → 平台审核 → 审核通过 → 上架到公开市场
                          ↓
                      审核拒绝 → 用户修改后重新提交
```

### 2.2 局域插件上架流程

```
企业开发者提交插件 → 企业管理员审核 → 审核通过 → 上架到企业市场
                              ↓
                          审核拒绝 → 开发者修改后重新提交
```

## 3. 权限申请审核流程

```
终端用户申请权限 → 企业管理员审核 → 审核通过 → 用户获得使用权限
                            ↓
                        审核拒绝 → 用户收到拒绝原因，可重新申请
```

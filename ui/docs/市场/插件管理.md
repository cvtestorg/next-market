## 1. 插件列表视图

| 视图 | 说明 | 显示内容 |
|------|------|----------|
| 已安装插件 | 当前环境已安装的插件 | 插件信息、运行状态、版本、操作按钮 |
| 未安装插件 | 市场中可安装的插件 | 插件信息、价格、安装按钮 |

## 2. 插件状态

| 状态 | 说明 |
|------|------|
| 未安装 | 插件未安装到当前环境 |
| 已安装-运行中 | 插件已安装且正在运行 |
| 已安装-已停止 | 插件已安装但已停止运行 |
| 待更新 | 插件有新版本可更新 |

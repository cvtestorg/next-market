# 插件搜索功能

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 搜索方式

| 搜索类型 | 说明 | 匹配字段 |
|----------|------|----------|
| 关键词搜索 | 根据关键词模糊匹配插件 | 插件名称、插件描述 |
| 作者搜索 | 根据作者名称搜索插件 | 作者名称 |
| 分类搜索 | 根据插件分类筛选 | 插件分类标签 |

## 2. 搜索场景

| 场景 | 描述 |
|------|------|
| 市场浏览 | 用户在插件市场中搜索可安装的插件 |
| 已安装管理 | 用户在已安装列表中搜索已安装的插件 |

## 3. 筛选条件

- 来源分类（官方/局域/个人）
- 付费类型（免费/收费）
- 组合筛选

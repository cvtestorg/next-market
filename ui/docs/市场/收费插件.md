# 收费插件

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 购买流程

```
用户浏览收费插件 → 点击购买 → 完成支付 → 获得使用权限 → 可安装使用
```

## 2. 收费模式

| 模式 | 说明 |
|------|------|
| 一次性购买 | 支付一次，永久使用 |
| 按期订阅 | 按月/年付费，到期需续费 |

## 3. 价格设置（插件作者）

| 设置项 | 说明 |
|--------|------|
| 收费模式 | 一次性/订阅 |
| 价格金额 | 具体价格 |
| 订阅周期 | 月/年（订阅模式） |

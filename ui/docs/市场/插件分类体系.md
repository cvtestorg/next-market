# 插件分类体系

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 按来源分类

| 分类 | 说明 | 审核方 | 上架条件 |
|------|------|--------|----------|
| 官方插件 | 由平台官方开发和维护的插件 | 平台 | 平台直接上架 |
| 局域插件 | 企业内部开发的插件，仅限本企业使用 | 企业管理员 | 企业管理员审核通过后上架 |
| 个人插件 | 终端用户开发的插件，可公开到市场 | 平台 | 平台审核通过后上架 |

## 2. 按付费类型分类

| 类型 | 说明 |
|------|------|
| 免费插件 | 无需付费，直接安装使用 |
| 收费插件 | 需要购买后才能使用 |

## 3. 分类组合

- 官方插件 - 免费
- 官方插件 - 收费
- 局域插件 - 免费
- 局域插件 - 收费
- 个人插件 - 免费
- 个人插件 - 收费

# 页面结构

## 文档版本
- 版本：1.0
- 更新日期：2025-01-10

---

## 1. 插件市场首页

- 搜索栏
- 分类筛选（来源、付费类型）
- 插件列表（卡片形式展示）
- 分页/无限滚动

## 2. 插件详情页

- 插件基本信息
- 作者信息
- 版本历史与更新日志
- 操作按钮区域
- 配置管理（Tab切换企业/个人配置）

## 3. 我的插件

- 已安装插件列表
- 权限申请记录
- 购买记录

## 4. 插件管理（开发者）

- 我上传的插件列表
- 版本管理
- 权限管理
- 数据统计

## 5. 企业管理（管理员）

- 局域插件管理
- 权限审批列表
- 企业配置管理
